# E-commerce Analytics – Power BI Dashboard

This repository contains the Power BI dashboard built for the e-commerce analytics project.

The dashboard focuses on:
- Revenue and sales trends over time
- Customer behavior (repeat vs one-time customers)
- Product category performance
- Impact of ratings and reviews on sales
- Key KPIs such as AOV, CLV, and conversion-related metrics
